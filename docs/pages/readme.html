Thank you for downloading Domain Portfolio Manager! :) Use the following guide to help you install it.

Make sure you read everything before you begin uploading files. That includes the license.


1.) Requirements

  • PHP 5.2.x or newer (5.2.17+ recommended)
  • MySQL 4.1.x or newer (5.1.57+ recommended)

2.) Installing/Upgrading

The following gives important information that you should read before any installation or upgrade. After INSTALLING or UPGRADING be sure to delete the ./install/ folder.

** ALWAYS DO A BACKUP BEFORE ATTEMPTING AN UPGRADE - BACKUP YOUR DATABASE AND FILES **

  • Note what you have for your db details in ./includes/config.class.php
  • Backup your templates, and any modified files.
  • Upload and overwrite all files.
  • Re-enter your database details in ./includes/config.class.php
  • Compare your modified templates/files to the new version, changing them as needed.

  • TO UPGRADE TO 1.0.x
    • Not needed yet :)

Please also remember, when uploading files to:

  • Upload all .php, .tpl, and .txt files in ASCII mode.
  • Upload all images in BINARY mode.


NEW INSTALL


  • Create a MySQL database.
  • Open ./includes/config.class.php and edit each of the following:
    • dbhost
    • dbuser
    • dbpass
    • dbname
  • You may also want to update the following in ./includes/config.class.php
    • dbemail
    • timezone
  • Upload all files and folders to your server, however, do not upload the /docs/ directory.
  • CHMOD the ./admin/backups/ directory to 0777.
  • Go to: http://www.example.com/path/to/install/ and install the script.
  • Once finished, be sure to delete the /install/ directory.
  • Also delete the /docs/ directory if you uploaded it.

UPGRADING


  • Please above in the TO UPGRADE FROM.

3.) Usage

  • Go to http://www.example.com/path/to/admin/
  • Enter the username and password you gave during the installation.
  • Then proceed to http://www.example.com/path/to/admin/site_config.php
  • After that.. proceed in adding domains :-)

4.) Warning

DO NOT EDIT ANYTHING UNLESS WHERE TOLD, OR UNLESS YOU KNOW WHAT YOU ARE DOING. IF YOU DO SO IT MAY RESULT IN THE SCRIPT NOT WORKING.
